
# Airq

A esp32 based sensor suite sending multiple environmental data points to a mqtt broker.


## Features

- co2 in ppm
- pm1 in ppl
- pm2.5 in ppl
- pm10 in ppl
- gamma radiation in micro Sievert
- gamma-rays per second
- voc air quality
- temperature 
- humidity
- pressure
